# Notebook-mini-application
Ce projet consiste à développer une application web complète permettant à un utilisateur de s'authentifier et de gérer ses notes personnelles (type « pense-bête »). Chaque note contient les informations suivantes : • Un titre (obligatoire, max 100 caractères) • Un contenu textuel  • Une date de création  • Un niveau de priorité
